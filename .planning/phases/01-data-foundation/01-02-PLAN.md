2. Create `InvoiceService.cls` -- `public with sharing class InvoiceService`:

   a. `getAccountIdForCurrentUser()` -- private static method:
      - Query: `SELECT ContactId, Contact.AccountId FROM User WHERE Id = :UserInfo.getUserId() AND ContactId != null LIMIT 1`
      - Return `users[0].Contact.AccountId` or null if empty/null

   b. `getInvoices(String tab, String dateFrom, String dateTo, Integer pageNumber, Integer pageSize)` -- `@AuraEnabled(cacheable=true)`:
      - Call `getAccountIdForCurrentUser()`, throw `AuraHandledException('Unable to determine account context')` if null
      - Calculate offset: `(pageNumber - 1) * pageSize`
      - Base WHERE: `(BillingAccountId = :accountId OR Sold_To_Account__c = :accountId) AND Status NOT IN ('Canceled', 'Error')`
      - Tab filters:
        - `'open'`: add `AND Balance > 0 AND Status = 'Posted'`
        - `'paid'`: add `AND (Status = 'Paid' OR Balance = 0)`
        - `'priorYear'`: add `AND InvoiceDate >= :priorYearStart AND InvoiceDate <= :priorYearEnd` where priorYear = `Date.today().year() - 1`
      - Date range filters: if dateFrom not blank, add `AND InvoiceDate >= :parsedDateFrom`; if dateTo not blank, add `AND InvoiceDate <= :parsedDateTo`. Parse dates with `Date.valueOf(dateFrom)`.
      - SELECT fields: `Id, DocumentNumber, InvoiceDate, DueDate, TotalAmountWithTax, Balance, Status, PO_Number__c, Portal_Status__c`
      - Append `WITH SECURITY_ENFORCED ORDER BY InvoiceDate DESC LIMIT :pageSize OFFSET :offset`
      - Execute with `Database.query()`
      - Run separate COUNT() query with same WHERE clause
      - Return `InvoiceListResult` with invoices, totalCount, pageNumber, pageSize

   c. `getInvoicesByIds(List<Id> invoiceIds)` -- `@AuraEnabled(cacheable=true)`:
      - Call `getAccountIdForCurrentUser()`, throw if null
      - Query: `SELECT Id, DocumentNumber, InvoiceDate, DueDate, TotalAmountWithTax, Balance, Status, PO_Number__c, Portal_Status__c FROM Invoice WHERE Id IN :invoiceIds AND (BillingAccountId = :accountId OR Sold_To_Account__c = :accountId) WITH SECURITY_ENFORCED`
      - Return `List<Invoice>`

   d. Inner class `InvoiceListResult`:
      - `@AuraEnabled public List<Invoice> invoices`
      - `@AuraEnabled public Integer totalCount`
      - `@AuraEnabled public Integer pageNumber`
      - `@AuraEnabled public Integer pageSize`
